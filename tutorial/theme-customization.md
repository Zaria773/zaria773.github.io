# 主题自定义

这里介绍如何火速根据你的酒馆主题或喜欢的图片搓一个好看的主题。

## 一、拷打ai
把1+2+3一起丢给哈基米
- 1.酒馆主题CSS/配色满意的图片
- 2..acu-modal-container,
.acu-wrapper,
.acu-app {
--acu-bg-nav: #e8f5e9;
--acu-bg-panel: #e8f5e9;
--acu-border: #c8e6c9;
--acu-text-main: #2e7d32;
--acu-text-sub: #81c784;
--acu-btn-bg: #c8e6c9;
--acu-btn-hover: #a5d6a7;
--acu-btn-active-bg: #43a047;
--acu-btn-active-text: #fff;
--acu-table-head: #dcedc8;
--acu-table-hover: #f1f8e9;
--acu-shadow: rgba(0, 0, 0, 0.1);
--acu-card-bg: #ffffff;
--acu-badge-bg: #dcedc8;
--acu-menu-bg: #fff;
--acu-menu-text: #2e7d32;
--acu-input-bg: rgba(255, 255, 255, 0.7);
--acu-overlay-bg: rgba(46, 125, 50, 0.2);
}
- 3. 参照颜色资料：酒馆主题css/我发的图片，与18色变量名与三个选择器的资料，提取并合理搭配颜色写个新的CSS样式

## 二、自调与保存
- 把ai吐出来的粘贴进自定义CSS，按转为滑块，不满意的地方可以在这里预览自调。调好以后配背景保存就完事了。
- ps：只调低一下卡片和表头背景透明度加上自定义背景就会挺好看了！
